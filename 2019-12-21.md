##调试脚本##
1. 使用选项`-x`,启用shell脚本的调试功能
2. 使用`set -x`和`set +x`对脚本进行部分调试，例如
~~~
   #! /bin/bash
   for i in {1..6};
   do
        set -x
        echo $i
        set +x
    done
    echo "Script end"
~~~

##tr的巧妙用法##
1. 求和：`cat sum.txt | echo $[ $(tr '\n' '+' ) 0 ]`,sum.txt的每一行都是一个数字
2. 去除空白行：`cat multi_blanks.txt | tr -s '\n'`

##sort的参数##
1. 合并两个已经排序的文件：`sort -m sorted1 sorted2`
2. 检查文件是否已经排序过：
```
#!/bin/bash
sort -C filename
if [ $? -eq 0 ]; then
    echo Sorted;
else
    echo Unsorted;
fi
```
3. 排序键的指定用`-k`参数，k指的是列数
